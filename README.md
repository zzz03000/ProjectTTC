## 📌 개요

ElemetalUP은(는) Unity로 개발된 블록 쌓기 장르의 게임입니다.
트리키 타워 클론을 가져와서 여러 기능을 추가 및 개발을 하였습니다.
이 프로젝트는 대학 친구들과 함께 팀으로 개발했으며, 저는 주로 시스템 프로그램을 담당했습니다.
